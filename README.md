# Sorting Algorithm Visualizer

Interactive visualization of 5 classic sorting algorithms built with vanilla HTML, CSS, and JavaScript.

🔗 **Live Demo:** https://aayushsolanki-1.github.io/sorting-visualizer

## Algorithms
- Bubble Sort — O(n²)
- Insertion Sort — O(n²)
- Selection Sort — O(n²)
- Merge Sort — O(n log n)
- Quick Sort — O(n log n) avg

## Features
- Real-time comparison and swap counters
- Adjustable array size (10–120 elements)
- Variable animation speed
- Color-coded states: comparing, swapping, sorted

## Tech Stack
HTML · CSS · Vanilla JavaScript
